# CthulhuDiceBot
Dice Bot for Call Of Cthulhu(TRPG).

日本語で動く、クトゥフル用ダイスbot

主なアクションは以下の3つです

- ダイスロール
    - 例）1d100でダイス振って
- 技能判定
    - 例）目標値50で目星
- 抵抗判定
    - 例）10と12で抵抗判定
